# GreetingCard 💌

Portfolio Order: 1️⃣

## Overview
GreetingCard is a beginner-friendly Jetpack Compose app focused on text UI, alignment, spacing, and simple theming. It demonstrates how to build a clean single-screen interface from scratch.

## What This App Demonstrates
- Basic Compose functions and composable structure
- Text styling using fonts, size, and color
- Layout composition with Column, Row, and modifiers
- Preview-driven UI development

## Key Features ✨
- Greeting message rendering
- Sender name display
- Clean static UI suitable for first Compose practice

## Tech Stack 🛠️
- Kotlin
- Jetpack Compose
- Android Gradle Plugin

## Project Structure
- app/ -> Android app module
- app/src/main/ -> Kotlin code + resources
- gradle/ -> Wrapper and dependency management files

## Run Instructions ▶️
1. Open this project folder in Android Studio.
2. Sync Gradle dependencies.
3. Start emulator or connect a phone.
4. Run the app.
