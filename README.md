## Common Tech Stack 🛠️

- Kotlin
- Android Studio
- Jetpack Compose
- Material 3 (selected projects)
- Gradle build system

## How To Run Any App ▶️

1. Open Android Studio.
2. Open one project folder.
3. Let Gradle sync finish.
4. Select emulator or physical device.
5. Click Run.
