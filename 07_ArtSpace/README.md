# ArtSpace 🖼️

Portfolio Order: 7️⃣

## Overview
ArtSpace is an artwork showcase app where users can browse pieces and see related information. It focuses on image presentation and navigation between content items.

## What This App Demonstrates
- Image-driven Compose UI
- Previous/Next interaction
- Metadata display for each art item

## Key Features ✨
- Artwork viewer
- Artist and title details
- Navigation controls for gallery flow

## Tech Stack 🛠️
- Kotlin
- Jetpack Compose
- Material 3

## Project Structure
- app/ -> Android app module
- app/src/main/java/ -> Gallery UI logic
- app/src/main/res/drawable/ -> Art assets

## Run Instructions ▶️
1. Open project in Android Studio.
2. Sync Gradle.
3. Run on emulator or phone.
