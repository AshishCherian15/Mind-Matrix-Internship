# Greeting Card

A simple Android greeting app that displays a message and sender in a styled layout.

## Features
- Static greeting message UI.
- Compose text and layout practice.

## Tech Stack
- Kotlin
- Jetpack Compose

## Project Structure
- app/ Android app module
- app/src/main/ source and resources

## How to Run
1. Open with Android Studio.
2. Allow Gradle sync.
3. Run on emulator/device.

## Requirements
- Android Studio
- JDK 17
- Android SDK
