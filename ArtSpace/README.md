# Art Space

An Android gallery-style app that shows artwork with title, artist, and navigation controls.

## Features
- Shows one artwork at a time.
- Previous and Next navigation.
- Clean Compose UI for image and metadata display.

## Tech Stack
- Kotlin
- Jetpack Compose
- Material 3

## Project Structure
- app/ main Android application module
- app/src/main/ source code and resources
- gradle/ Gradle wrapper files

## How to Run
1. Open this folder in Android Studio.
2. Sync Gradle dependencies.
3. Run on emulator or physical device.

## Requirements
- Android Studio
- JDK 17
- Android SDK
